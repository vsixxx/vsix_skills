# Scene Library

Reusable Creative Production scene packs live here.

Scene packs define commercial environments where a supplied product, service, venue, offer, or business asset can be placed. Skills should consume this library instead of keeping private copies in their own `references/` folders.

Current packs:

- `packs/business-point-of-sale-scenes.json`: 25 slot-based business, retail, service, customer, demo, pickup, and handoff scene archetypes.
