# Dashboard Map: Style Guide Adapter

Use `dashboard-builder` only when style-guide adaptation needs a review dashboard for many materials, deviations, or approvals.

## Decision Question

Which materials conform to the required style, and what deviations block release?

## Recommended Sections

1. Overview: `metric_strip`, `verdict`, `flags`.
2. Style Deviations: `wide_table`, `bar_chart`.
3. Materials Coverage: `source_readiness`, `table`.
4. Required Fixes: `action_register`.
5. Approval Path: `timeline`.

## Top KPIs

- Materials reviewed
- Critical style deviations
- Pages/files affected
- Brand/compliance blockers
- Fix owners
- Approval status

## Required Sources

- Style guide
- Materials under review
- Precedent deck or memo
- Reviewer comments
- Output manifest
