# 红黑白科技风格 配图提示词指南

## 风格定义

现代科技感插图风格，灵感来源于 Sider、ZenMux 等科技产品的视觉设计。
特点：红黑白三色配色、等距视角（Isometric 2.5D）、几何化设计语言。
适合：技术文章、产品推广、工作流程展示等场景。

## 配色方案（严格三色原则）

| 颜色 | 色值 | 用途 | 占比 |
|------|------|------|------|
| 红色 | #E63946 / #FF0000 | 强调、高亮、连接线、重要元素 | 20-30% |
| 黑色 | #000000 / #1A1A1A | 主体结构、人物剪影、文字 | 30-40% |
| 白色 | #FFFFFF | 背景、文字、留白 | 30-40% |
| 灰色 | #CCCCCC / #808080 | 仅次要元素（<10%） | <10% |

## 视角规范

- Isometric projection: 30° angle
- No vanishing point (parallel lines stay parallel)
- Consistent scale regardless of position
- Viewing angle: slightly elevated, 30-45°

---

## 几何化设计语言

- 平台/基座：矩形平台（rectangular platforms）
- 建筑/结构：立方体、长方体（cubes, rectangular blocks）
- 人物：简化剪影（simplified silhouettes）—— 无面部细节
- 图标：几何化图标（geometric icons）
- 连接线：电路风格线条（circuit-like lines）

## 布局模式

### 模式 A - 左中右三分法
```
Left (30%): Problem/Before scenario
Center (40%): Solution/Platform/Mascot
Right (30%): Result/After scenario
```

### 模式 B - 分层架构
```
Top layer: High-level components
Middle layer: Core processing
Bottom layer: Foundation/Infrastructure
```

### 模式 C - 工作流程
```
Left: Input/User A
Center: Platform/Processing
Right: Output/User B
Connection lines showing data flow
```

## 元素库

### 人物元素
- Developer/designer figure in black silhouette
- Sitting at desk with laptop/computer
- Standing and interacting with screens
- Minimal facial features, focus on posture

### 技术元素
- Laptop/computer screens showing data/code
- Server racks (isometric cubes)
- Data flow lines (red glowing circuits)
- Cloud icons (geometric clouds)
- Database cylinders
- API connection nodes

### 数据可视化
- Bar charts (3D isometric bars)
- Line graphs (on floating screens)
- Pie charts (isometric circles)
- Metric cards (floating rectangular cards)

### 装饰元素
- Geometric shapes floating (cubes, spheres, pyramids)
- Grid background (subtle, light gray)
- Glow effects on connection lines (red)
- Shadows for depth (subtle, gray)
