# Creative Production Offer Library

JSON-first reusable offer and product-advertising archetypes for Creative Production workflows.

Use `registry-index.json` to discover active packs. Packs live under `packs/` and use the same `version` plus `packs[]` shape as the ad and scene libraries. Each pack stores reusable creative families: archetypes, prompts, placeholders, constraints, and default instantiations without bundling rendered example media.

Rendered examples belong in `outputs/`, team docs, or an external approved media source rather than the launch plugin package.
