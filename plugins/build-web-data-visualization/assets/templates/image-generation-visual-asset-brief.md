## Negative Prompt / Exclusions

- No publication branding or copied layouts.
- No text, numbers, axes, logos, or source notes inside the generated image unless explicitly required.
- No misleading scale cues.
- No decorative detail that competes with data labels.
- No invented factual objects when the story requires real-world specificity.
- No visual choices that weaken the claim, hide caveats, or imply unsupported precision.

## Review Checklist

- The generated asset supports scale, place, mechanism, motion, or stakes.
- It remains legible under the chart and annotation layer.
- It does not imply unsupported precision or causality.
- It can be described accurately in alt text.
- It has a static fallback and a mobile crop.
- If used as a concept, it was shown as part of the large-screen and mobile concept set with concise plan and interaction bullets, then approved or iterated before project changes or implementation code.
- If approved as a concept, it has been converted into a binding semantic design contract before implementation.
- The final asset or implementation matches the contract or records approved meaning-preserving deviations.
