# Export Checklist

- Target medium is known.
- Figure sizes are fixed for each slot.
- Large-screen, mobile portrait, and optional mobile landscape export or screenshot sizes are fixed when the visualization is browser-facing.
- SVG, PNG, PDF, or HTML outputs are chosen intentionally.
- Fonts, colors, and label density fit the final medium.
- Captions and source notes are preserved.
- Mobile screenshots preserve the claim, caveat, source context, and main visualization without hover, autoplay, permissions, or a strong network.
- Stale/offline/partial-data states have exportable or screenshot-ready representations when relevant.
- Regeneration path is documented.
