# Export and Automation Sources

## What Problem This Solves

This index groups the primary references for producing PDFs, slide decks, and document-ready figure outputs.

## When to Use It

Use this when generating reports, PDFs, PowerPoint or Google Slides content, or reusable figure assets.

## Key Takeaways

- Browser-rendered HTML to PDF is often the most reliable route for styled reports.
- Slide automation is usually easiest when chart assets are generated first and placed second.
- Document automation should preserve a regeneration path so updated data can rebuild the same package.

## Common Mistakes

- Depending on screenshots when vector or deterministic exports are available.
- Building slide or PDF layouts before slot sizes, theme rules, and figure dimensions are defined.
